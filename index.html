Mais o que são tudo isso?

!DOCTYPE:

serve para ditar a versão 5 do HTML, com ela, você ja vem com uma estrutura montada certinha para você usar

HTML lang="":

O HTML serve para ditar que tudo que estiver dentro do HTML, esta em html, ja o lang serve para ditar o idioma do site, para deixar em portugues, basta colocarPT-BR

HEAD:

essa tag diz que é a cabeça do site, ou seja, tudo o que o usuarios não vai ver ou mexer, mais vai mudar o site, CSS ou os metas dados

Meta

as metas tags são metadados, aonde somente os navegadores leiram, as metas tags fica localizada dentro do HEAD, os usuarios não vem os metadados, somente os navegadores
